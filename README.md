# fastnum

Arbitrary precision, fixed-size signed and unsigned decimal numbers implemented in pure Rust.

## Features



### Serialization and Deserialization

The `serde` feature enables serialization and deserialization of `fastnum` decimals via the [`serde`](https://docs.rs/serde/latest/serde/) crate.

### Database ORM's support

The `diesel` feature enables serialization and deserialization of `fastnum` decimals via the [`diesel`](https://docs.rs/diesel/latest/diesel/) crate.

### Autodocs crates support

The `utoipa` feature enables support of `fastnum` decimals for autogenerated OpenAPI documentation via the [`utoipa`](https://docs.rs/utoipa/latest/utoipa/) crate.
